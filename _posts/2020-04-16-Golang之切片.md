# 切片

> 切片是长度可变的数组,由三部分组成:指针、长度、容量
> 指针:指向切片第一个元素的地址
> 长度:切片元素的数量
> 容量:切片开始到结束元素的数量

## 声明

> 切片声明需要指定组成元素的类型，但不需要指定存储元素的数量(长度),切片会被初始化为nil，表示不存在的切片

```go
package main

import "fmt"

func main() {
	var names []string
	// %t 比较条件输出true false
	fmt.Printf("%T, %t, %#V\n", names, names == nil, names)
}
```
结果:

    go run blog.go            
    []string, true, []
